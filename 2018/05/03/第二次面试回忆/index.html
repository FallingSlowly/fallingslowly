第二次面试回忆

发表于 2018-05-03 | 分类于 面经

第二次面试在中电软件园那里，一开始先做半个小时题目，然后hr面试十分钟，最后技术负责人面了半个小时左右。

笔试题目

选择题
  • public, protected, private作用范围
  • final定义方法
  • Integer定义变量
  • Collection和Map的区别
  • Spring的一些知识
问答题
  • 数组有length()方法吗？String有length()方法吗？
  • 接口和抽象类的区别（感觉问的频率好高，两次面试都问到了=、=）
  • list, set, map存取元素之间的区别
  • HashMap和ConcurrentMap的区别
  • 一道分布式的题目，不会，也忘记了。。。
手写SQL题目
  • 一共有四道，完全忘记各种SQL语句了。。。

面试题目

看一下你做的题目正确率，然后讲一下，接着问问题：

  • 讲一下你简历里面的实习和项目
  • Spring的控制反转和依赖注入
  • 用过Linux没有，说一下几个简单的命令
  • 讲一下Mybatis，ORM框架，作用是什么
  • Maven用过没
  • Git用过没
  • ArrayList和LinkedList的区别
  • 用过jQuery和Ajax没有
  • 对Java并发，分布式这些了解过没

总结

没有问的很深，也没有问一道算法题666666，感觉小公司和大公司的区别蛮大的。
Anyway，继续学习。
